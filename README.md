# Hangman Game
A simple text-based Hangman Game developed using Python.
## Features
- Random word selection
- 5 predefined words
- Maximum 6 wrong guesses
- Letter-by-letter guessing
- Win and Lose conditions
## Concepts Used
- Random Module
- While Loop
- If-Else Statements
- Strings
- Lists
## How to Run
Run the Python file using:
python main.py
